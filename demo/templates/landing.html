bot = Scraper(delay=0.5)

# Crawl every page, extract typed records
books = bot.crawl(
    "https://books.toscrape.com",
    item="article.product_pod",

        "price": "p.price_color::text",
    },
    next="li.next a::attr(href)",

The strategy-agnostic toolkit

One API. Every scraping strategy.

From a single static page to a JS-heavy SPA to a hidden JSON API—ByteCraw picks the right tool, or you choose explicitly.

Auto-strategy

Tries static HTML first, falls back to a real browser when a page is JS-rendered. No guessing.

API-first

Hit the JSON API behind a site and skip HTML entirely. Up to 2.5× faster than a headless browser.

LLM-ready

Convert any page to clean Markdown with one call. Strips noise, slashes token cost for agents.

strategies.py 
from bytecraw import Scraper

bot = Scraper()

# 1 · static HTML  (requests + BeautifulSoup)
page = bot.static("https://books.toscrape.com")

# 2 · dynamic JS  (real Chromium via Playwright)
page = bot.browser("https://quotes.toscrape.com/js", wait="div.quote")

# 3 · hidden API  (clean JSON, no parsing)
data = bot.api("https://quotes.toscrape.com/api/quotes").json()

# 5 · authenticated session  (login + CSRF token)
s = bot.session().login(url, data=creds, csrf_field="csrf_token")

# LLM · 2,755 tokens of HTML -> 423 tokens of Markdown
md = page.markdown()
